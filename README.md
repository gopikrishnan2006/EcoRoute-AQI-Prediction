# EcoRoute: Air Quality Prediction & Health Advisory

## Overview
This project predicts Air Quality Index (AQI) using machine learning and provides health-based recommendations.

## Features
- AQI prediction using Random Forest
- Data analysis and visualization
- Feature importance analysis
- Risk classification system

## Technologies Used
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn

## Key Insights
- PM2.5 has the highest impact on AQI
- Higher pollution leads to unhealthy conditions
